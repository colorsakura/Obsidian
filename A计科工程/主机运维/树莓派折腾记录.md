## 安装系统

```shell
dd bs=4M if=/path/to/os.img of=/dev/sda

# 挂载boot分区后，创建ssh文件
# mount /dev/sda1 /mnt
touch ssh

# wifi(2.5G) 自动连接，创建wpa_sup，内容如下
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
network={
  ssid="WIFI名"
  psk="WIFI密码"
}
```
